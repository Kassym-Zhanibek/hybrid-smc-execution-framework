# Hybrid SMC Execution Framework

Semi-automated MT5 execution and risk management framework for XAU/USD trading.

## Overview

This project automates trade execution and risk management while trade decisions remain discretionary.

The framework is designed for intraday trading and includes position management, risk controls and session-based execution logic.

## Features

- Position sizing
- Risk allocation
- Break-even management
- Trailing stop management
- Daily loss limits
- Drawdown protection
- Trading session controls

## Trading Process

Trade entries are identified manually using market structure and orderflow analysis.

The framework manages execution, position handling and risk controls automatically.

## Technologies

- MetaTrader 5
- MQL5
